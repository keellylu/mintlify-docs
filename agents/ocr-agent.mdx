## Properties
### Parameters

The `OCRAgent` accepts 1 optional argument: 

```
OCRAgent(model(Optional))
```

<ResponseField name="model" type="OCRModel" required>

### Example
Here is an example of the `OCRAgent` as part of a Workflow for license plate recognition.

```python example.py
image_path = "./examples/license_plate.jpg"
image = Image.open(image_path)
workflow = Workflow()
workflow.add_step(OCRAgent(model=TextractModel()))
workflow.execute(image)
```


## Use Cases
The `OCRAgent` is designed to handle use cases where extracting text from images is crucial:

> - Document Digitization
> - Automated Data Entry
> - Translation
> - Text-to-Speech
> - License Plate Recognition
