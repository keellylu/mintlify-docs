## Overview

There are 4 main classes of Agents in Overeasy: 
 - **Image Agents** - Operate on a single PIL image input
 - **Text Agents** - Operate on a single text input
 - **Detection Agents** - Operate on a Detection
 - **SplitAgents** - Splits a bounding box Detection into multiple child Nodes/Detections
 - **JoinAgents** - Joins multiple Detections into a single Detection
