## Overview

There are 4 main classes of Agents in Overeasy: 
- Image Agents
- Text Agents
- Detection Agents
- Join/Split Agents

### **Image Agents**

Image Agents operate on a single PIL image input.
 {/* ```mermaid
%%{init: {'theme': 'neutral' } }%%
flowchart LR
    direction TB
        subgraph NODE1
            direction LR
            I1(Image) ~~~ D1(Data)
        end
        subgraph NODE2
            direction LR
            I2(Image) ~~~ D2(Data)
        end
        NODE1 --ImageAgent--> NODE2
    
    style I1 fill:#ECECFD
``` */}

 ### **Split Agent** 
 Split Agents split a Bounding Box Detection into multiple child Nodes/Detections.

```mermaid
%%{init: {'theme': 'neutral' } }%%
flowchart TD
  A(Image, Detections) --- SplitAgent(SplitAgent) --> 1(Image Crop 1, Individual Detection) & 2(Image Crop 2, Individual Detection) & 3(Image Crop 3, Individual Detection)

  style SplitAgent fill:#ffffff,stroke:#333,stroke-width:0px
```
 
 ### **Join Agent** 
 Join Agents join multiple Detections into a single Detection.

 ```mermaid
%%{init: {'theme': 'neutral' } }%%
flowchart TD
  1(Image Crop 1, Individual Detection) & 2(Image Crop 2, Individual Detection) & 3(Image Crop 3, Individual Detection) --- JoinAgent(JoinAgent) --> 4(Image, Detections)

  style JoinAgent fill:#ffffff,stroke:#333,stroke-width:0px
```
